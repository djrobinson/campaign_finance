  • What is it
    • -
  • The operating expenditures treemap represents all funds spent by a given committee
  • - The overall size of the rectangle represents all funds spent by the committee. Each smaller rectangle represents the portion of funds that were given to just one recipient.
  • - Clicking each rectangle will show a transaction by transaction breakout of each expense.
  • - How to navigate it
  • - Hover over any square to see the preview in the toolbar above the treemap. This will show basic information like amount, recipient, and the purpose of the transaction.
  • - Click on a rectangle to see a detailed view of the transaction or recipient. The three levels of detail are:
  • - Committee Level – Shows a breakout of the recipients of the committee’s expenses.
  • - Recipient Level – Shows a breakout of each transaction to a given recipient
  • - Transaction level – Shows detailed info for a given transaction and provides a link to the FEC Doc in the lower right hand corner.
